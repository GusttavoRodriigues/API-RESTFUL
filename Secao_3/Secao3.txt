
EndPoints.
Métodos HTTP:
Os endpoints geralmente são associados a métodos HTTP específicos, como GET, POST, PUT e DELETE.
O método HTTP utilizado com um endpoint determina a ação a ser executada no recurso.
Por exemplo, o método GET é usado para recuperar informações, o POST para criar, o PUT para atualizar e o DELETE para excluir.
 
 
Um endpoint é a URL onde seu serviço pode ser acessado por uma aplicação cliente.
